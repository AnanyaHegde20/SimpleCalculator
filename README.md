# Simple Calculator

A lightweight and beginner-friendly calculator application that performs
basic arithmetic operations. This project is ideal for learners who want
to understand fundamental programming concepts such as user input
handling, functions, UI structure, and clean code practices.

## Features

-   Addition\
-   Subtraction\
-   Multiplication\
-   Division\
-   Input validation and error handling\
-   Clean and minimal user interface

## Project Structure

    simple-calculator/
    │── src/
    │   └── main code files
    │── assets/
    │── README.md

## Installation & Setup

1.  Clone or download the project\
2.  Open the folder\
3.  Run the application depending on your language/environment

## Usage

1.  Enter the first number\
2.  Choose an operation\
3.  Enter the second number\
4.  View the result
